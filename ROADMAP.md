# Roadmap

## Phase 0

- 프로젝트 골격 정리
- 디렉터리 구조 확정
- 문서 초안 작성

## Phase 1

- 인증 API 설계
- 사용자/경기/선수 모델 정교화
- Alembic 마이그레이션 초안 작성
- 오늘의 경주 및 경주 상세 수직 기능 완성
- 샘플 데이터 시드와 테스트 고정
- 결과/선수 이력/통계 수직 기능 확장
- CSV import CLI 추가

## Phase 2

- 데이터 수집 파이프라인 설계
- Redis + Celery 작업 분리
- 저장소/서비스 레이어 확정

## Phase 3

- 예측 모델 통합
- 분석 대시보드 구성
- 구독/결제 흐름 설계

## Phase 4

- 운영 배포 구성
- 모니터링/로깅 강화
- 보안 및 권한 정책 정리
